# Test Cases: Registration & Authorization

**Описание задания:**  
Созданы тест-кейсы для полного функционала интернет-магазина, включая регистрацию, авторизацию, работу с каталогом, корзиной, оплатой и историей заказов. 

**Цель:**  
Формализовать проверки на основе ранее разработанного чек-листа и подготовить их к системному тестированию всех ключевых модулей интернет-магазина

**Что было сделано:**  
- Разработано 60+ тест-кейсов  
- Описаны шаги и ожидаемые результаты  
- Использованы тестовые данные  
- Покрыты ключевые сценарии
